== Description ==

This plugin is an extension for [WPForms](https://wordpress.org/plugins/wpforms-lite/) to add an advance image upload field with support for drag and drop, multiple image upload functionality and image preview feature in the front end.

[youtube https://www.youtube.com/watch?v=CaMAVdjwnKk]

**Just Plug & Play**

* Goto WPForms form builder.
* Then find Multiple image uploads with preview section on bottom left.
* Drag Upload Image Button.
* Save the form.
* And you are done.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==


= Where can I see all uploaded Images? =

Go to your WordPress media Library, and you can find all your uploaded images from WPForms.
